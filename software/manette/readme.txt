les donnees de la manette sont envoyees en 5 int

le premier contient toutes les donnees des boutons:

LSB	R1
LSB-1	R2
LSB-2	L1
LSB-3	L2
LSB-4	Joystick Droit
LSB-5	Joystick Gauche
LSB-6	Channel

les 4 prochaines valeurs correspondent aux niveau des joysticks, dans l'ordre

Joystick Droit X
Joystick Droit Y
Joystick Gauche X
Joystick Gauche Y

les données sont envoyées en brut.
Le channel par défaut est à 0
